# Even-or-Odd-Checker
A Simple Python GUI application built with Tkinter to check whether a number is Even or Odd.
## Features
- Simple UI: Clean Tkinter interface, beginner friendly.
- Input Validation: Handles non-numeric input without crashing.
- Instant Feedback: Get results in one click.
- Zero Dependencies: Only uses Python's built-in Tkinter
- "Reset" button to clear input and output field
- "Check" button to show result   
## How To Run
1. Install Python (version 3.x)
2. Download or Clone this Repo
3. Open terminal in the project folder, then Run:
```bash
python "Even or Odd Checker.py"
```
